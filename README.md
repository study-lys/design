毕业设计使用说明书
**********************************************************************************************************************
1、前端

文件：demo
使用说明：
（1）进入demo文件夹
（2）运行 npm install 安装项目依赖
（3）依赖安装完毕运行 npm run serve 启动项目

到此  前端页面启动完毕 地址：http://localhost:8080/ （如果8080端口被占用，则会自动更换其他端口，具体看启动后的提示）
**********************************************************************************************************************
2、后端

文件：serve
使用说明：
（1）自行下载软件“xampp”（此软件集成apache以及mysql）
（2）安装，无脑下一步即可
（3）设置后端服务，启动  xampp  找到软件内  apache  ->   点击config ,在打开的下拉列表中选择第一个打开，在记事本中搜索‘DocumentRoot ’将它及它下一行的Directory  的路径改为  serve文件路径下的public（示例：‘C:\Users\Administrator\Desktop\design\serve\public）
（4）启动  xampp，start（启动）apache以及mysql。

到此 服务器启动完毕  地址：http://localhost:80/
**********************************************************************************************************************
3、数据库

文件：database
使用说明：
（1）将自己数据库的账号设置为 root    密码为空（不要密码）
（2）找到 database 下的 demo.sql
（3）新建数据库 demo 字符集 utf-8
（4）将demo.sql导入到 demo数据库中
